## 功能
- 播放在线音乐和本地音乐，支持多种音频格式。
- 对本地音乐音乐进行各种排序显示。
- 歌曲、专辑、艺术家、歌单等信息一目了然。
- 更改应用主题、背景和材质。
- 显示歌曲内嵌歌词，并含有更改字体、灵动词岛功能。
- 将歌曲添加到播放队列和播放列表中。
